## Aprendizado

* Utilização do plugin `path_provider` para salvar e ler arquivos na memória interna do dispositivo.
* Utilização de widgets como: *Dismissible*; *CheckboxListTile*; *SnackBar*; *RefreshIndicator*.
* Ordenação de listas.
* Utilização de `setState`.
